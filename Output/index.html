What's the difference between alert() and console.log()?

Alert() prints whatever is within the parentheses in a pop up box, and the user is required to hit 'OK' to continue. Console.log() prints whatever is within the parentheses to the console.

What's one situation where you might use an `alert()` to tell the user something?

Alerts are used to make sure information is seen by the user by requiring the user to hit 'OK' to proceed. Often these alerts will be some sort of warning. For example, if the user did not fill in all required fields of a form, an alert might pop up to give a warning.

What's one way that `alert()` might get obnoxious?

An alert could be obnoxious if it popped up everytime you went to the homepage.
